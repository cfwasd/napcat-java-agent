# Bot 功能指南

本文档面向 Bot 用户和运营者，介绍框架内置的所有可用功能。

---

## 一、修仙 RPG 系统

完整的群聊修仙 RPG 游戏，包含境界突破、渡劫、切磋战斗、宗门系统、丹药商店、双修等完整循环。离线期间也有修为收益。

### 1.1 入门：开启修仙

**触发方式**：`@bot /修仙`

首次修仙时，Bot 会为你随机生成三项天赋属性：

| 属性 | 范围 | 说明 |
|------|------|------|
| 根骨 | 5-15 | 修炼效率，影响修为获取速度 |
| 气运 | 5-15 | 影响渡劫成功率、奇遇概率 |
| 灵力 | 5-15 | 影响切磋战斗伤害 |

初始赠送 100 灵石。天赋一旦生成不可更改，但可通过奇遇事件永久提升。

### 1.2 境界体系

```
凡人 → 练气 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 大乘 → 真仙
```

- 每大境界含 4 小层：**初期(1) → 中期(2) → 后期(3) → 圆满(4)**
- **突破小层**：消耗修为，100% 成功
- **突破大境界**（圆满→下一大境界）：触发**天劫**

### 1.3 修炼与修为获取

三种修为获取方式：

**主动修炼**
- 触发：`@bot /修炼`
- 冷却：1 小时
- 收益：`根骨 × 境界系数`

**后台/离线收益**
- 两次交互之间自动计算时间差
- 公式：`时间差(小时) × 根骨 × 境界系数 × 0.5`
- 上限：最多累计 8 小时
- 查看：`@bot /修仙状态`

**签到联动**
- 触发：`@bot /签到`
- 修仙者签到额外获得：`根骨 × 2` 修为

### 1.4 突破与渡劫

**突破小层**
- 触发：`@bot /突破`
- 消耗指定修为值
- 修为需求随境界指数增长

**渡劫（大境界突破）**
- 触发：`@bot /渡劫`
- 天雷轮数：3~9 轮（境界越高轮数越多）
- 每轮成功率：`60% + 气运×2%`（上限 90%）
- 需**连续成功**所有轮数才算渡劫成功

**渡劫结果**
- 成功：踏入新大境界，修为清零重新累积
- 失败（80% 概率）：**转世重修**
  - 修为归零，境界重置凡人
  - 根骨保留 50%（向上取整）
  - 灵石和宗门关系保留
  - 获得「轮回印记」：修炼效率 +20%
- 失败（20% 概率）：重伤掉 1-2 大境界

**辅助丹药**
- **渡劫丹**：+20% 渡劫成功率（单次渡劫有效）
- **还魂丹**：渡劫失败时免于转世（仅重伤掉境）

### 1.5 意外事件

每次主动修炼时概率触发（基础 10%，气运每高 1 点 -0.5%）：

| 事件 | 概率 | 效果 |
|------|------|------|
| 走火入魔 | ~5% | 重伤状态，当前修为减半，修炼效率 -50%（持续 1 小时或吃疗伤丹恢复） |
| 天降祥瑞 | ~3% | 修为 ×3 暴击 |
| 上古遗迹 | ~1% | 随机属性永久 +1 |
| 无事发生 | ~91% | 正常修炼 |

### 1.6 切磋系统

**发起切磋**
- 触发：`@bot /切磋 @某人`

**应战**
- 触发：`@bot /应战`

**战斗规则**
- 伪回合制，双方轮流攻击
- 每人随机 3 个招式，招式含基础伤害和效果描述
- 伤害公式：`(招式基础伤害 + 灵力×2) × 随机波动(0.8~1.2)`
- HP 先归零的一方输
- 胜者 +10 声望，败者 +15 修为

### 1.7 宗门系统

**创建宗门**（需金丹期以上 + 500 灵石）
- `@bot /创建宗门 宗门名`

**加入宗门**
- `@bot /加入宗门 宗门名`

**宗门管理**
- `@bot /同意入宗 @某人`（宗主）
- `@bot /踢出宗门 @某人`（宗主）
- `@bot /退出宗门`
- `@bot /捐献 数量`（捐献灵石升级宗门）

**宗门效果**
- 灵气加成：宗门等级 × 2% 修炼效率
- 宗门满级 10 级，最多 50 人

**查看**
- `@bot /宗门状态`
- `@bot /宗门排行`

### 1.8 丹药商店

**查看商店**：`@bot /丹药商店`

| 丹药 | 价格（灵石） | 效果 |
|------|------------|------|
| 培元丹 | 50 | +修为(小)：根骨×境界系数×3 |
| 筑基丹 | 200 | +修为(中)：根骨×境界系数×8 |
| 疗伤丹 | 150 | 立即解除重伤状态 |
| 渡劫丹 | 300 | 渡劫临时 +20% 成功率 |
| 还魂丹 | 1000 | 渡劫失败免转世 |

**购买**：`@bot /购买 丹药名`

### 1.9 姻缘双修

**婚姻系统**
- `@bot /求婚 @某人`
- `@bot /同意求婚`
- `@bot /离婚`
- `@bot /我的CP`

**双修**
- 结婚后自动成为道侣
- 道侣加成：双方 +10% 修为获取（被动）
- 双修指令：`@bot /双修`
- 双修收益：`(双方根骨平均值 × 3) + (双方境界层数之和 × 2)`
- 冷却：24 小时

### 1.10 其他命令

| 命令 | 说明 |
|------|------|
| `@bot /修仙状态` | 查看修仙面板、离线收益 |
| `@bot /修仙排行` | 本群修仙者排行榜 TOP 10 |
| `@bot /修仙菜单` / `@bot /修仙帮助` | 显示完整菜单 |
| `@bot /签到` | 每日签到（修仙者额外修为） |
| `@bot /签到状态` | 签到记录和连签天数 |
| `@bot /运势` | 今日运势 |

## 二、群聊娱乐工具

框架内置 20+ 娱乐工具，全部通过 `@bot` 或唤醒词触发，无需额外配置。

### 2.1 游戏类

| 命令 | 说明 |
|------|------|
| `@bot 石头剪刀布` | 和 Bot 猜拳 |
| `@bot 猜数字` | Bot 想一个 1-100 的数字，你来猜 |
| `@bot 真心话大冒险` | 随机抽取真心话或大冒险题目 |
| `@bot 猜谜语` | 随机出一个谜语 |

### 2.2 占卜类

| 命令 | 说明 |
|------|------|
| `@bot 塔罗牌` | 随机抽取塔罗牌并解读 |
| `@bot 能不能/会不会/该不该...` | 魔力 8 号球是/否占卜 |
| `@bot 运势` | 今日运势 |

### 2.3 随机类

| 命令 | 说明 |
|------|------|
| `@bot 掷骰子` / `@bot roll d20` | 掷骰子，支持 d4/d6/d8/d10/d12/d20/d100，可带数量和修正值 |
| `@bot 帮我选 A 还是 B` | 选择困难症终结者，从选项中随机选择 |
| `@bot 分组 @A @B @C` | 将提到的人随机分成几组 |
| `@bot 惩罚转盘` / `@bot 大转盘` | 随机抽取一个惩罚/整蛊任务 |

### 2.4 文字类

| 命令 | 说明 |
|------|------|
| `@bot 讲个笑话` | 随机讲一个笑话 |
| `@bot 来个梗` / `@bot 表情包` | 推荐热梗/表情包 |
| `@bot 绕口令` | 随机出一个绕口令 |
| `@bot 文字特效 内容` | 对文字进行翻转、竖排、火星文等特效 |

### 2.5 漂流瓶

| 命令 | 说明 |
|------|------|
| `@bot 扔漂流瓶 内容` | 扔一个漂流瓶到海里 |
| `@bot 捞漂流瓶` | 随机捞一个别人的漂流瓶 |
| `@bot 漂流瓶统计` | 查看当前漂流瓶数量 |

### 2.6 实用小工具

| 命令 | 说明 |
|------|------|
| `@bot 计算 1+1` | 数学计算器 |
| `@bot 翻译 你好` | 多语言翻译 |
| `@bot 历史上的今天` | 查看今天历史上的大事 |
| `@bot 还有几天放假` / `@bot 下个节日` | 节假日倒计时 |
| `@bot 倒计时 2026-12-31` | 计算距离某日期还有多少天 |
| `@bot 编码/解码 base64 内容` | Base64/URL/Unicode 编码转换 |
| `@bot 现在几点` | 获取当前时间 |

## 三、AI Agent 能力

除上述固定功能外，Bot 还具备 AI Agent 能力，支持自然语言交互：

- **自然对话**：日常聊天、知识问答
- **工具调用**：自动识别意图并调用对应工具
- **多轮推理**：复杂任务可多次调用工具
- **图片理解**：支持图片输入分析
- **长期记忆**：跨会话记住你的偏好和信息（详见 [Agent 工具集 -> 记忆系统](./09-agent-tools.md#一记忆系统)）
- **定时任务**：可为你设置定时提醒或任务（详见 [Agent 工具集 -> 定时任务](./09-agent-tools.md#六定时任务)）
- **网络搜索**：通过 SearxNG 获取实时信息（详见 [Agent 工具集 -> 网络搜索](./09-agent-tools.md#二网络搜索)）
- **文生图**：根据文字描述生成图片（详见 [Agent 工具集 -> AI 文生图](./09-agent-tools.md#四ai-文生图)）
- **文件操作**：下载文件、管理本地文件（详见 [Agent 工具集 -> 文件操作](./09-agent-tools.md#五文件操作)）

> 所有 Agent 工具的完整参数与触发示例，请参阅 **[Agent 工具集](./09-agent-tools.md)** 文档。
